## Key Features

- **Chat Operations**: Real-time and streaming chat interactions
- **Agent Management**: Create and manage different types of agents
- **Audio Processing**: Transcription and text-to-speech capabilities  
- **File Operations**: Upload and manage files
- **Collection Management**: Organize and manage data collections
- **Administration**: Comprehensive admin endpoints for system management

## Authentication

The API uses OAuth2 Client Credentials flow for authentication. Different endpoints require different scopes:

```json
"securitySchemes": {
  "Oauth2ClientCredentials": {
    "type": "oauth2",
    "flows": {
      "clientCredentials": {
        "scopes": {
          "SuperAgent:write": "Write super agent changes",
          "SuperAgent:read": "Read super agent info",
          "Agent:write": "Write agent changes",
          "Agent:read": "Read agent info",
          "Template:write": "Write template changes",
          "Template:read": "Read template info"
        },
        "tokenUrl": "/api/admin/auth/token"
      }
    }
  }
}
```
