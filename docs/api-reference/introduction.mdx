## Quick Start

1. **Get your API key** - Sign up for an account and generate your API key
2. **Make your first request** - Test the API with a simple GET request
3. **Explore endpoints** - Browse all available endpoints in the next section

<Note>
  All API requests require authentication. Make sure to include your API key in the request headers.
</Note>
